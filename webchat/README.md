# WebChat
A fully browser-hosted, ephemeral, minimalist chatroom system.  
No accounts. No storage. No databases.  
Rooms only exist while the host keeps their tab open.

---

## ‚ú® Features
- Host a chatroom directly in your browser  
- Generate a room code that others can join  
- No login or signup required  
- Anonymous mode for privacy  
- Messages exist only in memory and auto-delete after 1 hour  
- All messages disappear permanently when:
  - The host closes the page  
  - The room is closed  
- Peer-to-peer communication via WebRTC  
- Minimalist UI with gray chat bubbles  
- Fully embeddable in any website  
- Open-source and MIT licensed
